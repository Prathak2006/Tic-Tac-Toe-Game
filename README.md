# Tic Tac Toe Game 

A Tic Tac Toe game built using HTML, CSS, and JavaScript.

##  Features

- Two-player game (Player O and Player X)
- Turn-based gameplay with visual turn indicator
- Score tracking for both players
- Reset Game option (resets board only)
- New Game option (resets full game and scores)
- Match system (play multiple rounds)
- Input field to select number of matches
- Start button to begin the game
- Displays current match and total matches
- Shows final winner after all matches are completed
- Clean and interactive user interface

##  Tech Stack

- HTML
- CSS
- JavaScript

##  How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
